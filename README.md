# Smart-Energy-Usage-Analyzer

## Overview

A Python data analysis project that cleans messy customer energy data, engineers new features, and generates visualizations to answer business questions. This project demonstrates a complete data analysis workflow using industry-standard Python libraries.

## Features

* Clean missing values and placeholder data
* Engineer new analytical features
* Perform exploratory data analysis (EDA)
* Create bar charts, histograms, and scatter plots
* Analyze customer usage, billing, and regional trends

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib

## Skills Demonstrated

* Data Cleaning
* Feature Engineering
* Data Visualization
* Exploratory Data Analysis (EDA)
* Business Data Analysis
